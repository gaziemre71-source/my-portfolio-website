# Building the PDF Eliminator Tool

Welcome to the deep dive of my PDF parsing automation system! In this project, I aimed to completely replace standard, slow, manual PDF copy-pasting tasks with a **Streamlit** + **PyMuPDF** powered interface.

## Challenge
The primary challenge was that real-world PDFs are messy:
- Inconsistent table structures
- Scanned documents with no text layer
- Scattered metadata

## Architecture & Solution

I designed a 3-tier system to tackle this problem effectively:
1. **Extraction Layer**: Utilizing PyMuPDF to extract text blocks and spatial coordinates.
2. **Analysis Layer (LLM)**: Sending the raw structured text to an LLM via API to classify the document type and extract meaningful key-pairs.
3. **UI Layer**: A Streamlit dashboard where users can instantly visualize the extracted payload before saving it to the database.
