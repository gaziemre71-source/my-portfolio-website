# 🚀 %100 AI-Assisted Freelancer Portfolio

Welcome to my personal portfolio website! This project is a modern, high-performance, and visually stunning demonstration of my work and skills, built with a unique "AI-first" approach.

## 🛠️ Tech Stack
This site avoids heavy frameworks in favor of a fast, modern "LOC" (Low Complexity) stack:
- **[HTMX](https://htmx.org/)**: Powers the Single Page Application (SPA) experience with seamless AJAX transitions without complex JavaScript.
- **[Alpine.js](https://alpinejs.dev/)**: Handles reactive UI components and state management with minimal overhead.
- **[Tailwind CSS](https://tailwindcss.com/)**: Provides a "Neon Dark" design system with rapid utility-first styling.
- **[Marked.js](https://marked.js.org/)**: Dynamically renders blog posts from Markdown files.

## ✨ Key Features
- **100% AI-Assisted Development**: Every line of code and design decision was guided or generated by advanced AI.
- **Neon Dark Aesthetics**: A premium, high-contrast UI designed for maximum visual impact.
- **Dynamic Projects**: Data-driven project showcase loaded via JSON.
- **Markdown Blog**: Easy-to-maintain blog system.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## 🚀 Quick Start
To run this project locally, simply serve the root directory using any local web server (e.g., Live Server in VS Code, Python's `http.server`, or Nginx).

```bash
# Example using Python
python3 -m http.server 8000

# Or using Node.js (npx)
npx serve .
```
